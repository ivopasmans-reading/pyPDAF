This module calls PDAFomi iso_C_binding wrappers.

Function calls convert Python objects for Fortran routines. Detailed information on the PDAFomi properties or subroutines can be found at https://pdaf.awi.de/trac/wiki/OverviewOfOMIRoutines and http://pdaf.awi.de/trac/wiki/ImplementFilterAnalysisOverview
